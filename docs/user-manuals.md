# User Manuals

Comprehensive PDF user manuals for Voice Notepad. Each manual is versioned to match the application release.

## What's in the Manual

The user manual includes:

- **Installation guide** - Setup instructions for Windows, Linux (AppImage, Debian, tarball)
- **Getting started** - API key configuration and first transcription
- **Main interface** - Detailed walkthrough of all tabs (Record, History, Cost, Analysis, Models, About)
- **Settings** - Configuration options for API keys, audio, behavior, and hotkeys
- **Keyboard shortcuts** - In-app and global hotkey reference
- **Troubleshooting** - Common issues and solutions

## Manual Updates

Manuals are updated with each major feature release. The version number corresponds to the application version the manual covers.
